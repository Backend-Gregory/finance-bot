# 💰 MoneyTrackerBot — Telegram бот для учёта доходов и расходов

Бот помогает вести личный финансовый учёт: добавлять доходы и расходы, смотреть статистику, выгружать отчёты в CSV, Excel и Google Sheets.

## 📋 Возможности

- ➕ **Добавление транзакций** (доход/расход, категория, описание)
- 📊 **Статистика** за всё время, месяц, неделю, сегодня
- 📁 **Экспорт отчётов** в CSV, Excel, Google Sheets
- 🔒 **Защита от спама** (rate limiting)
- 🛡️ **Глобальный обработчик ошибок**
- 📝 **Логирование** в файл

## 🚀 Технологии

- Python 3.13
- Aiogram 3.x (FSM, роутеры, middleware)
- SQLAlchemy 2.0 (ORM, SQLite)
- Google Sheets API (gspread)
- CSV / Excel export (openpyxl)

## Примечание

Бот сделан для портфолио, пока не задеплоен, поэтому публичной ссылки на работающего бота нет.
